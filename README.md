# 🖼️ Image Processing & Computer Vision 

This repository contains a comprehensive collection of image processing techniques, ranging from traditional mathematical enhancement to modern deep learning architectures.

---

## 📂 Project Directory

### 1. 🔍 [Traditional Image Processing](./Traditional_Image_Processing)
Focuses on spatial domain enhancements and structural segmentation.
* **Techniques:** Histogram Equalization, Global Thresholding, and Contour Analysis.
* **Segmentation:** Region Growing and Splitting algorithms for object isolation.

### 2. 🧠 [Deep Learning & CV](./Deep_Learning_and_CV)
Advanced applications using neural networks and dimensionality reduction.
* **Neural Networks:** CNN implementations for image classification and the MNIST digit dataset.
* **Feature Engineering:** Principal Component Analysis (PCA) and image patching for dataset preparation.

---

## 🛠️ Technical Stack
* **Language:** Python
* **Vision Libraries:** OpenCV, Scikit-Image
* **Deep Learning:** TensorFlow/Keras or PyTorch
* **Data Tools:** Matplotlib, NumPy, Pandas
